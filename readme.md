# インストール

Releasesのv1.0より、使用するPCのcpuに合わせたzipファイルをダウンロードして下さい。この方法でダウンロードを行う場合は環境構築は不要です。

# 実行

zip解凍後に、distディレクトリ内のgui_app.pyまたはfcs_gui.pyを実行して下さい。
セキュリティにブロックされた場合は、設定よりブロックを解除してから再度実行して下さい。

# 利用方法

詳しい利用方法は[こちらのREADME](https://github.com/kensuketakahama/FCS_RICS_Analysis_Takahama)を確認して下さい。
RICSとFCSのディレクトリ内に詳しい操作方法がREADMEにまとめられています。また、修正・開発を行う場合もこのリポジトリをローカルにクローンして行うようにして下さい。
